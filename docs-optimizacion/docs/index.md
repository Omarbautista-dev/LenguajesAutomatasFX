# Optimización de Código - LenguajesAutomatasFX

Este sitio documenta el proceso de análisis y optimización aplicado al proyecto **LenguajesAutomatasFX**, desarrollado en JavaFX para la materia de Lenguajes y Autómatas II.

El objetivo principal fue mejorar la calidad del código, reducir redundancias, detectar posibles problemas de rendimiento y documentar los cambios realizados mediante herramientas profesionales de análisis.

## Herramientas utilizadas

- SonarQube for IDE
- VisualVM
- Analizador incluido en IntelliJ IDEA

## Objetivo

Mostrar evidencia del código antes y después de la optimización, explicando qué problema existía, qué herramienta ayudó a detectarlo y qué mejora se aplicó.
